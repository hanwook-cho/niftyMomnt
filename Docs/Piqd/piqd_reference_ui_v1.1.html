Screen 01
Snap Viewfinder — Rest (Layer 0)
Maximum simplicity. Shutter (clean ring) and mode pill only. Full-screen camera behind Dynamic Island.
Screen 02
Snap Viewfinder — Layer 1 (Single Tap)
Zoom pill (673pt), aspect ratio pill, flip button, gear ⚙ top-left (→ Inbox / Settings), unsent badge, invisible level line. All elements clear of shutter.
Screen 03
Snap — Sequence Capture (4 of 6 frames)
Yellow shutter ring, Dynamic Island State 1 (4 dots filled), Safe Render crop guide. Layer 1 chrome hidden.
Screen 04
Roll Viewfinder — Rest (Layer 0)
Grain overlay, warm grade, light leak (top-right), film canister shutter, OCR-A counter, stopped-down aperture.
Screen 05
Roll Viewfinder — Roll Full (0 shots remaining)
Shutter at 40% opacity (non-responsive). "Roll's full. See you at 9." overlaid on live viewfinder. Mode pill still active.
Screen 06
Snap — Sequence Strip Preview
Floating 28% overlay (safeH×0.28≈213pt) at bottom. 6-frame contact sheet auto-loops. Text-link send/save. Swipe up → circle selector.
Screen 07
Snap — Drafts Tray Sheet
3 items: still (no timer), sequence (no timer), clip (amber pill badge <1h). UNSENT label + large count. CLEAR ALL right. Text-link actions per row.
Screen 08
Inbox — Received Snaps
No content preview — asset type only. Yellow unread dot (not a count badge). Timer only when <3h remaining.
Screen 09
Film Archive — Moment List
Narrative cards. Full-width hero image, Moment label, ambient detail, friend avatars. Received Roll shows attribution.
Screen 10
Roll Unlock Reveal — 9PM Ritual
Darkroom surface. Current asset (photo 4 of 18). Opened friends (amber border) vs waiting (dim). Reactions + reply. Quiet, not celebratory.
Screen 11
Mode Switch — Long-Hold In Progress (Snap → Roll)
User holding mode pill ~70% through 1.5s arc. Pill border glows amber. Clockwise arc fills around pill. Shutter visible but locked (dimmed). Release before arc completes = abort, no sheet.
Screen 12
Mode Switch — Confirmation Sheet (Snap → Roll)
Arc completes → sheet slides up. "Switch to Roll?" heading. Stopped-down aperture symbol in amber. SWITCH CTA (amber fill). "Stay in Snap" text-link. Tap outside = dismiss, no switch.
Screen 13
Snap — Circle Selector Sheet
.medium detent sheet over live viewfinder. "Send to" header + asset type subtitle. 5 friends horizontal — 2 selected (yellow border), 1 nearby dot. Yellow SEND → button. Ephemeral policy note below.
Screen 14
Asset View — Received Snap (Still)
Full screen. Minimal header: ← inbox / sender name / "viewing now". Asset fills screen behind header. Reaction chips + reply field above home indicator. Viewed trigger fires on appear.
Screen 15
Film Archive — Moment View
Pushed from list. Header: Moment label + ambient metadata. Friends row (opened/unopened dots). Vertical asset scroll. Per-asset tap → share/save bar (3s fade). “Save all to Photos” batch at bottom. FR-ARCHIVE-10/11/12, FR-SHARE-SOCIAL-03/04.
Screen 16
Storage Warning — First Roll Capture
Contextual modal blocks first Roll Mode shutter tap. “Roll photos live in Piqd only.” Dismissible but not skippable on first Roll. Two actions: “Got it — start shooting” (amber CTA, dismiss + proceed) and “Export to Photos after each unlock” (sets preference). FR-STORAGE-08.
Screen 17
Sender — Response Thread Sheet
Sheet over sender’s Snap viewfinder. Triggered by notification tap after receiver reacts or replies. Asset thumbnail + “Jiyeon reacted haha” + text reply. Sender reply field (one level only). Purge note below. FR-SHARE-RECV-18/19, §5.4.
Screen 18
Recipient — Purged State
Full screen. Shown if recipient navigates back to an already-viewed (purged) asset, or 24h hard-ceiling expiry. Minimal dark, centered copy. No asset, no chrome. “Piqd has left the chat.” FR-SHARE-RECV-17.
Screen 19
Snap — Format Selector Open (Layer 2)
Triggered by swipe-up on shutter or long-press shutter. Format pill: [Still] [Sequence] [Clip active—red] [Dual]. Layer 1 chrome visible behind. Shutter morphed to Clip (red ring + square). Collapses on selection or 3s idle. FR-SNAP-FORMAT-01/02/03/04.
Screen 20
Snap — Clip Recording Active
Hold shutter → recording begins. Red arc fills clockwise (~46% at 7s of 15s). Timer “0:07” above shutter. Inner square shrinks. Layer 1 chrome hidden. Mode pill dimmed (mode switch disabled). Release or ceiling → S7 Clip Ready. FR-SNAP-CLIP-01/02/03/04.
Screen 21
Snap — Still Post-Capture Confirmation
150ms transient state after still capture and send. Shutter ring shows checkmark (delivered) or arrow (queued). No thumbnail ever. Camera stays live. Auto-returns to S0 Rest in 150ms. Pending variant shown inset. FR-SNAP-NO-THUMB-02, §3.7.
Screen 22
Roll — Unlock Imminent (30min to 9PM)
Within 30min of 9PM. Counter shifts amber + breathes (4s pulse). DI State 2: circular fill arc + "9PM" label. Can still shoot if counter >0. §4.4 R6 state.
Screen 23
Roll — 9PM Developing Overlay (R7)
9PM fires. Grain doubles. "Developing your Roll…" centered over viewfinder. DI State 3: film-strip bars + "Developing…". No progress bar. StoryEngine + iCloud running in background. §4.6 R7 state.
Screen 24
Roll Viewfinder — Layer 1 (Single Tap)
Single tap on viewfinder. Zoom pill (amber active) + flip button + gear ⚙ top-left appear. Film sim three-dot indicator bottom-left. Gear → Inbox / Settings menu. Auto-retreats 3s idle. §4.2 R1 state.
Screen 25
Film Archive — Received Roll (Receiver's View)
Receiver's Moment view. "from Jiyeon" attribution in amber below Moment label. Same layout as Screen 15 otherwise. "permanent · no expiry" badge. Friends row shows sender + recipients. FR-ARCHIVE-12 receiver side.
Screen 26
Snap — Dual Capture Format (Idle)
Dual selected from format selector. Shutter: red ring + split diagonal circle. Front PiP inset top-right. No flip button (both cameras active simultaneously). FR-SNAP-DUAL-01/02/04/07.
Screen 27
Roll — Film Sim Change (R2 State)
Short edge swipe (<80pt) on left 20% while in Layer 1. Preset label ".fujiCool" appears bottom-left, fades 1.5s. Dots update. Viewfinder shifts to cool blue tint. §4.3 R2 state.
Screen 28
Roll — Live Photo Format (Shutter Morph)
Long edge swipe (>80pt with upward component) selects Live format. Shutter: film canister base + concentric dashed pulse ring + amber indicator dot. "Live" label appears 1.5s. §4.3 R3 state, FR-ROLL-LIVE-01.
Screen 29
Trusted Circle — QR Invite Screen
Light background. 200×200pt QR code centered. "Invite your first friend" heading. "Share invite link instead →" in yellow. "START SHOOTING →" CTA. Max 10 friends, QR or link only. §7.4, §9 Trusted Circle.
Screen 30
Settings
iOS UITableViewController. Accessed via long-hold mode pill. Sections: Capture, Roll Mode, Snap Mode, Storage (with archive warning note), Circle, About. §8 Settings spec.
