Change log · v1.7
A
Top Bar height + icon size
Height 56pt → 72pt. Icon pills 30×22pt → 40×32pt. Glyphs 11px → 15px. Roll counter 11px/500 → 16px/600. Film icon reinforces the roll metaphor for new users.
B
Low-light indicator NEW
Pulsing amber dot + "Low light — camera active" glass pill. Confirms the camera is alive in dark scenes — prevents the dark viewfinder from reading as a crash or freeze state.
B
Mode ghost label + swipe dots
Ghost opacity 0.28 → 0.42. Label size 11px → 13px. Dots 4pt/5pt → 5pt/7pt. Improves discoverability of swipe-left / swipe-right mode switching.
C
Preset Bar taller + bolder text
Height 36pt → 48pt. Preset name 12px/600 → 15px/700. Accent dot 6pt → 9pt. "Hold to change" hint 10px / opacity 0.30 → 13px / 0.55. Passes glare contrast.
D
Shutter button — larger
Outer ring 68pt → 96pt ⌀. Button body 54pt → 78pt ⌀. Count label 13px → 18px/700. Ring stroke 2.5pt → 3pt. Exceeds 44pt min tap target by a wide margin.
D
Safe area coverage
Zone D background extends +34pt below the shutter (home indicator safe area). Eliminates the bare #0A0A0A strip visible on iPhone 15/16 at the bottom edge.
D
Flip hint + thumbnail
"Double-tap to flip" 9px / opacity 0.28 → 12px / 0.50. Thumbnail 52×36pt → 64×46pt with LIVE badge. Both changes pass WCAG AA large-text threshold.
📐 Responsive — 6.1″ to 6.9″

All zones use flex layout with no hardcoded pixel heights except the safe-area inset (env(safe-area-inset-bottom) in production). The shutter row uses align-items: center + dynamic bottom padding so the layout reflows correctly across all supported device sizes.
